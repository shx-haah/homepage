  • Making assumption about the allowable fff: Assume fff lies in some set of functions F\mathcal{F}F. This is somewhat to making assumptions about PPP (this is obtained from the computation of R∗(P)R^*(P)R∗(P)).

    • Large class of F\mathcal{F}F -- it is hard to find the best fff.
